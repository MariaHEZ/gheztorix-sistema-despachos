# gheztorix-sistema-despachos
Aplicación desarrollada en Java utilizando programación orientada a objetos para la gestión de despachos, incluyendo manejo de expedientes, documentos y operaciones de facturación electrónica.
# Sistema Digital para la Gestión de Despachos (GHEZTORIX)

## 📌 Descripción
GHEZTORIX es un sistema digital orientado a optimizar la administración de despachos mediante la gestión de clientes, expedientes, documentos y facturación electrónica.

El sistema permite consultar, emitir, cancelar y descargar facturas, así como gestionar información de manera eficiente desde distintos dispositivos como computadoras, laptops, tabletas y teléfonos móviles.

## 💻 Tecnologías utilizadas
- Lenguaje: Java
- IDE: IntelliJ IDEA
- Paradigma: Programación Orientada a Objetos

## ⚙️ Requisitos
- Java JDK 17 o superior
- IntelliJ IDEA (recomendado)

## ▶️ Ejecución del proyecto
1. Clonar el repositorio
2. Abrir en IntelliJ IDEA
3. Ejecutar la clase `Main.java`

## 📂 Estructura del proyecto
- Cliente.java
- Usuario.java
- Documento.java
- Expediente.java
- Factura.java
- Main.java

## 🧠 Conceptos aplicados
- Encapsulación
- Abstracción
- Modularidad
- Relaciones entre clases

## 📎 Diagrama de clases
El diagrama se encuentra en la carpeta `/docs`.

## 🔧 Configuración adicional
No se requieren configuraciones especiales adicionales, únicamente contar con el JDK instalado y correctamente configurado en el sistema.
